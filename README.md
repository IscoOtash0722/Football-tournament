# Football-tournament
Программа написанной в языке Python, это программа автоматический вычисляет очки, игры, голы и разница мяча каждой команды, если вводить туда название и счет команд. Так же автоматически сортирует команд по порядке убывание. В начале программы будет пункты меню чтобы выбрать действие такие как: Добавить и удалить команд, показать таблицу/сетку и тд
